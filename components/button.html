Button 按钮 ​

常用的操作按钮

基础用法 ​

可以使用 type、plain、round 和 circle 来定义按钮的样式

<template>
    <div class="basic">
        <div class="common">
            <Button>Default</Button>
            <Button type="primary">Primary</Button>
            <Button type="success">Success</Button>
            <Button type="warning">Warning</Button>
            <Button type="info">Info</Button>
            <Button type="danger">Danger</Button>
        </div>
        <div class="plain">
            <Button plain>Default</Button>
            <Button type="primary" plain>Primary</Button>
            <Button type="success" plain>Success</Button>
            <Button type="warning" plain>Warning</Button>
            <Button type="info" plain>Info</Button>
            <Button type="danger" plain>Danger</Button>
        </div>
        <div class="round">
            <Button round>Default</Button>
            <Button type="primary" round>Primary</Button>
            <Button type="success" round>Success</Button>
            <Button type="warning" round>Warning</Button>
            <Button type="info" round>Info</Button>
            <Button type="danger" round>Danger</Button>
        </div>
        <div class="circle">
            <Button icon="fa-regular fa-user" circle></Button>
            <Button type="primary" icon="fa-regular fa-pen-to-square" circle></Button>
            <Button type="success" icon="fa-regular fa-heart" circle></Button>
            <Button type="warning" icon="fa-regular fa-star" circle></Button>
            <Button type="info" icon="fa-regular fa-envelope" circle></Button>
            <Button type="danger" icon="fa-regular fa-trash-can" circle></Button>
        </div>
    </div>
</template>
<script setup>
    import Button from '@/components/Button/button.vue'
</script>
<style scoped>
    .basic {
        display: flex;
        flex-direction: column;
        gap: 15px;
    }
</style>

调整尺寸 ​

通过给按钮添加 size 属性来定义按钮大小
size 属性接受三个值 large、default、small

<template>
    <div class="basic">
        <div class="common">
            <Button size="large">Large</Button>
            <Button size="default">Default</Button>
            <Button size="small">Small</Button>
            <Button size="large" icon="magnifying-glass">Search</Button>
            <Button size="default" icon="magnifying-glass">Search</Button>
            <Button size="small" icon="magnifying-glass">Search</Button>
        </div>
        <div class="round">
            <Button size="large" round>Large</Button>
            <Button size="default" round>Default</Button>
            <Button size="small" round>Small</Button>
            <Button size="large" icon="magnifying-glass" round>Search</Button>
            <Button size="default" icon="magnifying-glass" round>Search</Button>
            <Button size="small" icon="magnifying-glass" round>Search</Button>
        </div>
    </div>
</template>
<script setup>
    import Button from '@/components/Button/button.vue'
</script>
<style scoped>
    .basic {
        display: flex;
        flex-direction: column;
        gap: 15px;

Button API ​

Button 属性 ​

属性名说明类型默认值
type按钮类型primary success warning info danger-
size按钮尺寸large default smalldefault
plain是否为朴素按钮booleanfalse
round是否为圆角按钮booleanfalse
circle是否为圆形按钮booleanfalse
disabled是否禁用按钮booleanfalse
nativeType原生按钮类型NativeTypebutton
autofocus是否自动获得焦点booleanfalse
icon按钮图标的类名string—
loading是否显示加载状态booleanfalse
