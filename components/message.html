Message 消息提示 ​

基础用法 ​

<template>
    <div class="message-demo">
        <Button @click="open">Show Message</Button>
        <Button @click="openVNode">Show VNode</Button>
    </div>
</template>
<script setup>
import { h } from 'vue'
import { createDkMessage } from '@/components/Message/method.ts'
import Button from '@/components/Button/button.vue'

const open = () => {
    createDkMessage({
        message: 'This is a message.',
        onClose: () => {
            console.log('close')
        },
    })
}

const openVNode = () => {
    createDkMessage({
        message: h('p', { style: 'line-height: 1; font-size: 14px' }, [
            h('span', null, 'Message can be '),
            h('i', { style: 'color: teal' }, 'VNode'),
        ]),
    })
}
</script>
<style scoped>
.message-demo {
    display: flex;
    gap: 10px;
}
</style>

不同状态 ​

用来显示「成功、警告、消息、错误」类的操作反馈。

<template>
    <div class="message-demo">
        <Button @click="open('success')">Success</Button>
        <Button @click="open('warning')">Warning</Button>
        <Button @click="open('info')">Message</Button>
        <Button @click="open('error')">Error</Button>
    </div>
</template>
<script setup>
import { createDkMessage } from '@/components/Message/method.ts'
import Button from '@/components/Button/button.vue'

const open = (type) => {
    createDkMessage({
        message: `This is a ${type ?? ''} message`,
        type,
    })
}
</script>
<style scoped>
.message-demo {
    display: flex;
    gap: 10px;
}
</style>

Plain ​

        <Button @click="open('error')">Error</Button>
    </div>

</style>

可关闭消息 ​

设置 showClose 将显示关闭图标，点击可以提前关闭消息提示,默认情况下，dk-message 的 duration 为 3000 毫秒，你可以将它设置为 0 来表示不会自动关闭

<template>
    <div class="message-demo">
        <Button @click="open('success')">Success</Button>
        <Button @click="open('warning')">Warning</Button>
        <Button @click="open('')">Message</Button>
        <Button @click="open('error')">Error</Button>
    </div>
</template>
<script setup>
import { createDkMessage } from '@/components/Message/method.ts'
import Button from '@/components/Button/button.vue'

const open = (type) => {
    createDkMessage({
        message: `This is a ${type ?? ''} message`,
        type,
        showClose: true,
    })
}
</script>
<style scoped>
.message-demo {
    display: flex;
    gap: 10px;
}
</style>

自定义图标 ​

通过设置 icon 来自定义消息前面的图标

<template>
    <div class="message-demo">
        <Button @click="open">Show Icon Message</Button>
    </div>
</template>
<script setup>
import { createDkMessage } from '@/components/Message/method.ts'
import Button from '@/components/Button/button.vue'

const open = () => {
    createDkMessage({ message: 'This is a message.', icon: 'paper-plane' })
}
</script>
<style scoped>
.message-demo {
    display: flex;
    gap: 10px;
}
</style>

Message API ​

Message Attribute ​

属性名说明类型默认值
message消息提示string|VNode-
duration消息持续时间number3000
showClose是否可以关闭消息booleanfalse
type消息类型MessageTypeinfo
offsety 轴偏移量number20
plain是否开启 plain 样式booleanfalse
icon自定义消息图标string-
onDestory消息关闭后的回调()=>any-

Message Events ​

在使用 DkMessage 时，将方法作为属性传递给 DkMessage 方法

事件名说明类型
onShow消息提示开始显示时的回调()=>void
onClose消息提示自动关闭时的回调()=>void
onClickClose消息提示手动关闭时的回调()=>void
