Input 输入框 ​

用于输入字符

基础用法 ​

<template>
    <div class="input-demo">
        <Input v-model="iptval" placeholder="Place Input"/>
    </div>
</template>
<script setup>
import { ref } from 'vue'
import Input from '@/components/Input/Input.vue'

const iptval = ref('')
</script>
<style scoped>
.input-demo{
    width: 220px;
}
</style>

将 disabled 属性添加到 dk-input 使输入框进入禁用状态

<template>
    <div class="input-demo">
        <Input v-model="iptval" placeholder="Place Input" disabled/>
    </div>
</template>
<script setup>
import { ref } from 'vue'
import Input from '@/components/Input/Input.vue'

const iptval = ref('')

.input-demo{
    width: 220px;
}
</style>

带图标的输入框 ​

可以通过给 append suffix preffix prepend 插槽添加不同的图标来来让输入框带有图标

<template>
    <div class="input-demo">
        <Input
            v-model="iptval"
            placeholder="Place Input">
            <template #prefix>
                <Icon icon="magnifying-glass"></Icon>
            </template>
        </Input>
    </div>
    <div class="input-demo">
        <Input
            v-model="iptval"
            placeholder="Place Input">
            <template #suffix> 
                <Icon icon="magnifying-glass"></Icon>
            </template>
        </Input>
    </div>
</template>
<script setup>
import { ref } from 'vue'
import Input from '@/components/Input/Input.vue'
import Icon from '@/components/Icon/Icon.vue'

const iptval = ref('')
</script>
<style scoped>
.input-demo {
    width: 220px;
    margin: 8px 0;
}
</style>

通过 prepend 和 append 插槽可以实现输入框与其他内容的复合

复合型输入框 ​

Http://
.com 

<template>
    <div style="display: flex; flex-direction: column; gap: 15px">
        <Input
            v-model="iptval"
            placeholder="Place Input">
            <template #prepend> Http:// </template>
        </Input>
        <Input
            v-model="iptval"
            placeholder="Place Input">
            <template #append> .com </template>
        </Input>
        <Input
            v-model="iptval"
            placeholder="Place Input">
            <template #prepend>
                <Icon icon="magnifying-glass"></Icon>
            </template>
        </Input>
        <Input
            v-model="iptval"
            placeholder="Place Input">
            <template #append>
                <Icon icon="magnifying-glass"></Icon>
            </template>
        </Input>
    </div>
</template>
<script setup>
import { ref } from 'vue'
import Input from '@/components/Input/Input.vue'
import Icon from '@/components/Icon/Icon.vue'


const iptval = ref('')
</script>
<style scoped>
.input-demo {
    width: 220px;
}
</style>

Input API ​

Input Attribute ​

属性名说明类型默认值
v-model/modelValue输入框绑定的值string-
type输入框类型，支持原生类型stringtext
disabled是否禁用输入框booleanfalse
clearable是否显示一键清空按键booleanfalse
showPassword是否显示密码显示切换图标booleanfalse
placeholder原生属性string-
realonly是否只读booleanfalse
autocomplete原生属性string-
autofocus原生属性booleanfalse
form原生属性string-
inputStyle输入框样式CssProperties-

Input Events ​

事件名说明类型
input输入时触发(value:string):void
change输入框值改变且失去焦点时触发(value:string):void
focus输入框聚焦时触发(value:string):void
blur输入框失去焦点时触发(value:FocusEvnet):void
clear清空输入框时触发(value:FocusEvnet):void

Input Expose ​

属性名说明类型
ref输入框或文本域的 DOM 节点()=>void
clear清除输入框的值()=>void
focus使输入框获取焦点()=>void
blur使输入框失去焦点()=>void
select选中输入框中的文本()=>void
